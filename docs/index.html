Mr.F3RDI

A custom penetration testing framework with essential cybersecurity tools.

Inspired by Mr. Robot — reimagined and rebuilt by Ferdi Birgül.

Features

Information Gathering

Collect metadata about hosts, services, and users.

Password Attacks

Crack passwords and generate custom wordlists.

Wireless Testing

Analyze and attack Wi-Fi networks.

Exploitation Tools

Service-based system and network exploitation.

Sniffing & Spoofing

Intercept and spoof network traffic.

Web Hacking

Exploit common CMS vulnerabilities and inject payloads.

Private Web Hacking

Access protected files, backends, and databases.

Post Exploitation

Extend control after initial compromise.

bash <(wget -qO- https://raw.githubusercontent.com/Ferdibrgll/Mr.F3RDI/main/install.sh)
